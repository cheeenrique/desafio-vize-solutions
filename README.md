## 💻 Projeto

🚧 Um desafio realizado em React com as funções de Registrar, Logar e Exibir uma lista de usuários. 🚀

## 🔨 Recursos

- [x] Styled Components
- [x] Material UI
- [x] Eslint
- [x] Prettier
- [x] Context API
- [x] Axios
- [x] Formik
- [x] Yup

## ✨ Tecnologias

- [x] ReactJS

## 🏃‍♂️ Executando o projeto

Clonando o repositório

Use **yarn** ou **npm** install para instalar as dependências do projeto.
Em seguida, inicie o projeto.

```cl
npm start
```
